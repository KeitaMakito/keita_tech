# このブランチで上げた制作物について

+ 0624banksy は、javascriptを用いています。それぞれの番号をクリックすると、番号ごとの画像とテキストが表示されます。
+ Bootstrap_site1 は、Bootstrapを用いています。
+ javascript+googlemap は、googlemapとjavascriptを用いています。googlemap上で矢印キーを用いて画像を動かすことが出来ます。また、左側メニューの地名をクリックすると、その場所にジャンプ出来ます。
+ keita_jQuery_and_xml は、localhost上のApacheサーバと接続して、画像や文字を出力しています。
+ responsive_site は、クリエイターを目指す学生向けの、偉人の名言などを集めたレスポンシブデザインになっているHTMLページです。
+ keita_ajax は、MySQLとjavascriptを用いて、リアルタイム検索を実装しています。

###### keita_ajax と keita_jQuery_and_xml/ の動作環境はlocalhostです。
###### 他は、全て動作環境はWindowsです。
